# JB Process & Tech Consultoria

> Site institucional da **JB Process & Tech Consultoria** — consultoria e treinamentos em Lean, 5S, Kaizen e Lean Six Sigma, aliados a tecnologia própria para transformar processos industriais.

🌐 **Site ao vivo:** [jbprocesstech.com.br](https://www.jbprocesstech.com.br)

---

## 📋 Sobre o Projeto

Site institucional desenvolvido em **HTML, CSS e JavaScript puro**, hospedado via **GitHub Pages**. O objetivo é apresentar os serviços, aplicativos e diferenciais da JB Process & Tech de forma profissional e moderna.

---

## 🚀 Tecnologias Utilizadas

- **HTML5** — estrutura semântica
- **CSS3** — variáveis, grid, flexbox, animações
- **JavaScript** — interações e renderização dinâmica
- **Google Fonts** — Syne (títulos) + DM Sans (corpo)
- **GitHub Pages** — hospedagem estática gratuita

## 🔧 Como Atualizar o Site

1. Faça as alterações desejadas no arquivo `index.html`
2. Acesse o repositório no GitHub
3. Clique em **Add file → Upload files**
4. Faça o upload do `index.html` atualizado
5. Clique em **Commit changes**
6. Aguarde **1–2 minutos** — o site atualiza automaticamente
